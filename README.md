# Harmonic Starcove V17｜和谐星旅 · Draw + Sing Core

V17 聚焦两个最核心的问题：

1. 创作要更直观、更可操作：可以手绘线条转旋律，四轨道在手机和电脑上都能用。
2. 唱歌要既能训练，也能解释原理：训练器可以折叠，原理路径按需展开。

核心句子仍然是：

> 音乐是我们建构的爱的方程式。

## V17 主要新增

### 1. 手绘线条转旋律

创作页新增“画旋律”区域：

- 横轴 = 时间
- 纵轴 = 音高
- 平滑线 = 连贯歌唱
- 低谷 / 断点 = 换气或停顿

操作：

1. 在画布上画一条线
2. 点击“线条转旋律”
3. 系统会把线条量化成旋律网格
4. 可继续点击“更自然”加入呼吸、经过音和回家感

### 2. 四轨道手机/电脑自适应

创作页新增轨道切换：

- Melody 主旋律：表达
- Chord 和弦：情绪颜色
- Bass 低音：稳定中心
- Drum / Djembe 鼓圈：身体节奏

电脑端可以看整体轨道概览，手机端可以用当前轨道编辑 + 总览条，避免多轨道挤爆屏幕。

### 3. 歌唱训练器

歌唱页新增训练器，可直接使用，也可展开看原理。

训练器包括：

- 音准训练：目标音中心线 + 当前声音光点
- 气息训练：稳定河流式强度曲线
- 节奏 / 换气：强拍、弱拍、换气点
- 声带门：漏气 / 挤压 / 平衡闭合
- 情感表达：强弱、淡出、滑音、咬字

音准训练支持浏览器麦克风。允许权限后，可以显示当前频率和偏高/偏低反馈。

### 4. 原理按需展开

唱歌页不再一次性堆知识。现在是：

- 上方：训练器
- 下方：可折叠的原理路径
- 需要时展开：姿态、呼吸、声带、SOVT、声道、元音、换气、强弱、情感
- 还有风格比较：流行、美声、泛音、民歌 / 吟唱

### 5. 当前旋律和唱歌对齐

歌唱页可以播放当前创作旋律的机器哼唱 guide：

- hum / la / ma / ah / oo
- 标出建议换气段落
- 帮助把“写曲子”和“唱出来”连接起来

### 6. 修复多轨道 undefined

修复 trackRoles 渲染中 `undefinedMelody` / `undefinedBass` 的问题，轨道名称现在有稳定 fallback。
