# Harmonic Starcove V18｜和谐星旅 · Interaction Core

V18 聚焦“层层递进的交互学习”：

> 先操作 → 看反馈 → 展开原理 → 调整表达 → 继续探索

核心句子仍然是：

> 音乐是我们建构的爱的方程式。

## V18 主要更新

### 1. 手绘线条参数化

创作页的“画旋律”加入画笔参数：

- 平滑度：更像连贯歌唱，或更像折线跳动
- 灵敏度：小变化是否会变成明显音高变化
- 吸附：音阶内 / 半音 / 自由
- 线条：圆滑连线 / 折线断奏 / 点状跳音

线条转换时会根据这些参数生成旋律、换气点、停顿和少量半音装饰。

### 2. 多轨道颜色统一

轨道颜色统一：

- Melody 主旋律：金色
- Chord 和弦：粉紫
- Bass 低音：青蓝
- Drum / Pulse 鼓点：绿色
- Vocal Guide 人声：白金色

五线谱主要显示 Melody 与 Vocal Guide，轨道总览显示完整歌曲结构。

### 3. 模板系统

极简创作增加模板：

- 睡前
- 专注
- 勇气
- 悲伤
- 慈悲
- 鼓圈
- Ambient 星云
- 柔和钢琴

模板会自动生成：

- 音阶
- BPM
- 乐器
- 旋律轮廓
- 和弦
- 低音
- 鼓点
- 机器哼唱音节

### 4. 唱歌训练器压缩说明

唱歌页默认优先显示训练器。  
原理路径和风格比较都折叠，用户需要时再展开。

### 5. 唱歌评分

训练器加入：

- 音准分
- 稳定分
- 气息分
- 节奏分

打开麦克风后，音准训练会更新当前频率、偏高/偏低和评分条。

### 6. 发声器官动画重做

唱歌原理中加入“发声器官四层”图：

- 身体层：姿态 / 肋骨 / 气流
- 声源层：声带 / 起声 / 闭合
- 滤波层：咽腔 / 口腔 / 鼻腔 / 舌位 / 唇形
- 表达层：咬字 / 强弱 / 滑音 / 换气

重点解释：共鸣不是神秘词，而是声道形状改变频谱和 Formant。

### 7. 声音实验室可操作控件

声音实验室每个实验加入更明确的控件：

- 听变化
- 转成创作
- 用于唱歌
- 用于疗愈

并对频率、振幅、谐波、共振、节奏、噪声、张力等实验增加专用参数。

### 8. 星盘字体放大

星盘标签字体放大，核心点更清楚。  
小知识点仍然建议通过点击星系/右侧展开，而不是全部铺在图上。
