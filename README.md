# Harmonic Starcove V12｜和谐星旅

Musicalife / Harmonic Starcove 的第十二版：保持星河、函数、曼陀罗、比例创作的浪漫风格，但把今天的目标收束为四件事：

1. 能完成一段曲子
2. 能开始学习歌唱
3. 能寻找知识
4. 能通过动画学习和互动

核心句子保留为：

> 音乐是我们建构的爱的方程式。

## V12 主要更新

### 1. 星空点亮反馈
- 点亮的知识星星越多，背景星空会更丰富、更亮、更有光泽感。
- 点亮后会重新生成更明亮的 cosmos 背景。
- 保留“宇宙温柔呼唤”的星空体验。

### 2. 创作工作室
- 创作页改成三栏工作室：
  - 左侧：我要表达什么 / 音乐参数
  - 中间：旋律网格 + 简谱 + 五线谱雏形
  - 右侧：乐曲结构动画 + 曼陀罗 + 导出
- 模块使用可折叠区域，电脑端不再全部挤在一起。
- 情绪模板从文字改成图示卡：
  - 专注
  - 宁静 / 冥想
  - 快乐 / 幸福
  - 悲伤 / 安抚
  - 勇气
  - 爱的方程式
- 新增乐曲结构动画：
  - Intro
  - Motif
  - Repeat
  - Variation
  - Turn
  - Climax
  - Return
- 导出仍支持：
  - WAV
  - MIDI
  - MusicXML
  - JSON
  - PNG
  - 手机壁纸 PNG

### 3. 歌唱工作室
- 人声页改为循序渐进的 10 步课程：
  1. 姿态
  2. 呼吸波
  3. 声带门
  4. 起声
  5. SOVT
  6. 声道滤波
  7. 元音与 Formant
  8. 音阶
  9. 换气与乐句
  10. 合唱
- 每一课都有：
  - 原理
  - 动画
  - 练习
  - 常见错误
  - 声音反馈
- 人声示范声音改为更柔和的音色，避免尖锐刺激。

### 4. 声学实验库
- 声学实验从 3 个扩展为 12 个入口：
  - 频率与音高
  - 振幅与响度
  - 相位与抵消
  - 谐波与音色
  - 频谱与傅里叶
  - 共振与放大
  - 拍频与紧张感
  - 噪声颜色
  - BPM 与身体唤醒
  - 节奏密度
  - 音阶与情绪色彩
  - 和弦张力与解决

### 5. 知识星图
- 星图增加学习路径筛选：
  - 全部
  - 听见声音
  - 歌唱身体
  - 创作旋律
  - 选择音乐
  - 聆听音乐
  - 表达自我
- 每颗知识星点点击后会出现小课：
  - 原理
  - 动画
  - 怎么用
  - 下一步
  - 跳转实验 / 歌唱 / 创作

### 6. 音乐疗愈与生活
- 图书馆与生活模块加入方向：
  - 失眠
  - 认知
  - 情绪
  - 注意力
  - 疼痛
  - 慈悲心
- 强调音乐如何通过低干扰、低噪声、可预测结构和温和身体节律支持疗愈与冥想。
