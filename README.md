# Harmonic Starcove V15｜和谐星旅 · Visual Flow / 极简操作版

V15 基于 V14 继续优化，重点不是增加更多页面，而是让用户可以选择：

- **极简模式**：少讲解，快速操作，适合手机和直接创作。
- **教学模式**：显示原理、流程图、结构解释和调整建议，适合学习。

核心句子仍然是：

> 音乐是我们建构的爱的方程式。

## V15 主要更新

### 1. 进入页模式选择

进入网站时可以直接选择：

- 极简创作
- 歌唱练习
- 星图学习
- 选择音乐
- 生命共振
- 完整教学

也可以切换中文 / English 和低动效。

### 2. 极简 / 教学模式

创作页和歌唱页都支持模式切换：

- 极简：更少文字，直接操作、播放、导出。
- 教学：显示音乐参数、轨道分工、和弦、结构动画、作品分析。

### 3. 创作页更适合直接操作

极简创作保留：

- 情绪
- 音阶
- 乐器
- 长度
- BPM
- 旋律网格
- 播放 / 生成 / 重置 / 导出
- 曼陀罗和简短分析

教学模式保留：

- 情绪如何转成音乐参数
- Melody / Bass / Chord / Pulse 四轨道分工
- 和弦选择
- 乐曲结构动画
- 创作后分析
- WAV / MIDI / MusicXML / JSON / PNG / 壁纸导出

### 4. 歌唱页加入总流程动画

歌唱页顶部新增“从感受到歌声”的流程动画：

```text
感受 → 身体 → 呼吸 → 声带 → 声道 → 咬字 → 强弱/节奏 → 歌声
```

极简模式只显示核心练习；教学模式显示完整路径。

### 5. 星图可以点击展开分区

星图现在支持点击星云区域切换分区。右侧会显示当前分区的核心星点和下一步入口。

### 6. 多画布动画修复

之前一个页面多个 canvas 时，动画可能互相取消。V15 改为每个 canvas 单独管理动画循环，流程图、教学图、曼陀罗可以同时正常显示。

如果线上仍显示旧版，请确认 `index.html` 中引用的是 `harmonic-starcove-15`，并清理缓存或重新 deploy。
