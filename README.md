# Harmonic Starcove V16｜和谐星旅 · Music Maker Core + Singing Studio

V16 继续保留“音乐是我们建构的爱的方程式”的核心世界观，但把重点放到两个最核心的能力：

1. **更自然地创作一段小曲子**
2. **把写出的旋律对齐到歌唱练习**

这一版不是大而全扩展，而是让“创作 / 唱歌 / 原理”更容易被操作、听见、理解。

## V16 主要更新

### 1. 更自然的创作

创作不再只是单轨道格子音。

新增与强化：

- 多轨道默认可用：
  - Melody 主旋律
  - Bass 低音中心
  - Chord 和声颜色
  - Pulse 节奏推动
  - Djembe 非洲鼓身体节奏
- “更自然 / Humanize”按钮：
  - 自动加入呼吸休止
  - 自动改善结尾回家感
  - 播放时加入轻微时间偏移、力度变化
  - 大跳时加入经过音，让旋律不那么像像素块
- 新增音阶：
  - C blues / natural color
  - G major
- WAV 导出会包含多轨道安排。
- 仍保留 MIDI / MusicXML / JSON / PNG / 壁纸导出。

### 2. 机器哼唱 Vocal Guide

新增轻量版机器唱歌 guide，不追求真实 Vocaloid，但可以帮助用户：

- 把旋律用 hum / la / ma / ah / oo 唱出来
- 听见旋律的歌唱方向
- 理解哪里是一句、哪里适合换气
- 把创作页和歌唱页对齐

Vocal Guide 是通过 Web Audio 合成的柔和人声-like guide，不依赖服务器。

### 3. 歌唱教学更清楚

歌唱页新增更具体的 Singing Coach 数据与练习路径：

- 姿态：身体是乐器外壳
- 呼吸支持：气流不是越多越好
- 声带门：漏气 / 挤压 / 平衡
- SOVT：吸管、唇颤、哼鸣的回压原理
- 声道滤波：嘴型和舌位改变 Formant
- 咬字：辅音是边界，元音是流动
- 换气与乐句：换气在结构边界处
- 强弱与情感：不是更大声，而是更有方向

并且歌唱页可以直接播放当前创作旋律的机器哼唱 guide。

### 4. 手机极简操作修复

- 极简创作页布局更适合手机。
- 主要操作按钮做成底部 sticky 操作区。
- 控件改成更清楚的紧凑布局。
- 可快速：选择情绪 → 选择音阶 / 乐器 → 点格子 → 生成 → 更自然 → 播放 / 哼唱 / 导出。

### 5. 非洲鼓 / 身体节奏

创作里加入 djembe-like 低音与 slap 节奏，帮助理解：

- 强拍 = 身体重心
- 弱拍 = 流动
- 切分 = 惊喜
- 鼓点 = 身体的时间结构
